# Exact Reference Cases v1.0

## Purpose

Exact reference cases are small CTMC or graph-process examples with analytically known behavior.

They exist to verify that ATGS computations match known mathematical results before larger benchmark claims are made.

## Initial Case Families

1. two-state absorbing CTMC
2. three-state absorbing CTMC chain
3. reversible two-state CTMC
4. bidirectional three-state ring
5. unidirectional three-state ring

## Benchmark Principle

Exact reference cases should be small, inspectable, and mathematically transparent.
